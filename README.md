# TradeX - Twitter-Based Trading Bot

A simple trading bot that trades Tesla (TSLA) stocks based on sentiment analysis of tweets from:
- Elon Musk (@elonmusk)
- Tesla (@Tesla)
- Donald Trump (@realDonaldTrump)

## Project Structure

- `server/` - Backend services for tweet monitoring, sentiment analysis, and trading logic
- `client/` - Frontend dashboard (if needed) for monitoring bot activity

## Features

- Real-time tweet monitoring from key influencers
- Sentiment analysis of tweets
- Automated trading decisions based on sentiment
- Risk management and position sizing
